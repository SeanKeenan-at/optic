Optic API tokens are generated at [app.useoptic.com](https://app.useoptic.com). Click **Log in to create a token**, which will take you through the login flow.

![Log in to create a token](/img/docs/app.useoptic.com.png)

Once logged in, we'll generate an API token for you. Keep this token secure: This is how the GitBot will be authenticated to accept specifications for your API project.

![A token generated on app.useoptic.com](/img/docs/app.useoptic.com-token.png)
